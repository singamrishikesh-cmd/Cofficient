# 📘 Coefficient – Smart Vocabulary Learning App

## 🚀 Overview

**Coefficient** is a modern vocabulary learning web application designed to help users improve their vocabulary effectively by learning **5 new words daily**.

It uses an interactive **swipe-based learning system** to make learning engaging, fast, and easy to remember.

---

## 🎯 Problem Statement

Traditional vocabulary learning methods are:
- Boring 😴
- Hard to remember ❌
- Not interactive ❌

---

## 💡 Solution

Coefficient solves this by introducing:
- 📅 **Daily learning system (5 words)**
- 👉 **Swipe right = Known**
- 👈 **Swipe left = Review**
- 📊 **Progress tracking**
- ♾️ **Infinite learning mode**

---

## ✨ Key Features

- 🧠 Learn 5 new words every day
- 🎴 Swipe-based interactive UI
- 📈 Real-time progress tracking
- 🔁 Restart learning session
- ♾️ Infinite word stream mode
- 🎯 Simple and clean user experience

---

## 🖼️ How It Works

1. The app loads **5 words daily**
2. User interacts using swipe:
   - 👉 Right swipe → *Got it*
   - 👈 Left swipe → *Review*
3. System tracks performance
4. Displays completion and stats
